📝 Note App
Modern ve kullanıcı dostu bir not alma uygulamasıdır. Kullanıcıların fikirlerini ve notlarını organize bir şekilde saklayabileceği, Markdown desteği ile zengin içerik oluşturabileceği, etiketlerle kategorize edebileceği interaktif ve responsive bir web uygulamasıdır.

🛠️ Kullanılan Teknolojiler
React 19
TypeScript
Vite
Tailwind CSS
Redux Toolkit
Redux Persist
React Router DOM
React Select
SimpleMDE (Markdown Editor)
React Markdown
React Toastify
Lucide React
UUID

⭐ Özellikler
📝 Markdown destekli not oluşturma ve düzenleme
🏷️ Etiket (tag) sistemi ile notları kategorize etme
🔍 Başlığa ve etiketlere göre güçlü filtreleme
💾 LocalStorage ile kalıcı veri saklama (Redux Persist)
📱 Responsive ve mobil uyumlu tasarım
⚡ Vite ile optimize edilmiş yüksek performans
🎨 Tailwind CSS ile modern ve şık arayüz
🔔 Toast bildirimleri ile kullanıcı geri bildirimi
✨ Akıcı animasyonlar ve geçiş efekleri
🌙 Temiz ve minimalist kullanıcı arayüzü
📊 Toplam not sayısı ve filtrelenmiş not sayısı gösterimi
🔄 Real-time filtreleme ve arama
